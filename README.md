Python Setup
-----------------------

Using [Anaconda (a.k.a. Conda)](https://docs.conda.io/en/latest/):
```shell
conda env create -f environment.yml
```

Using `pip` with Python `>=3.8`:
```shell
pip install -r requirements.txt
```
